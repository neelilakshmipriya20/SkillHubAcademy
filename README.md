Project Title: Skill Hub Academy — A Responsive Educational Website
Tech Stack: HTML5, CSS3, Bootstrap 5
Tools: Visual Studio Code, GitHub Pages
----------------------------------------------------------------------------------------------------------------------------------
Step-by-Step Development Process
-----------------------------------------------------------------------------------------------------------------------------------------
1. Project Setup

Created a structured folder setup with index.html, about.html, and contact.html.

Added a separate folder assets/img/ to organize all project images.

2. Bootstrap Integration

Used Bootstrap 5 CDN for fast integration.

Explored the Bootstrap Documentation and Examples to understand navbars, cards, hero sections, and responsive layouts.

3. Home Page (index.html)

Designed a Hero Section introducing “Skill Hub Academy”.

Added a Feature Section explaining why users should choose the platform.

Included an Animated Testimonial Section using the AOS (Animate On Scroll) library.

Added a Footer with social links and branding.

4. Courses Page (about.html)

Added three course cards for:

Full Stack Web Development

Artificial Intelligence & Machine Learning

Cloud Computing

Used responsive card components and optimized images.

5. Contact Page (contact.html)

Created a simple and clean contact form using Bootstrap forms.

Added labels, placeholders, and a full-width submit button.

6. Styling and Optimization

Used consistent spacing, shadow effects, and color schemes.

Added custom CSS for hero background and typography.

Ensured all images were properly resized and placed in assets/img/.

7. Deployment

Uploaded the code to GitHub.

Deployed using GitHub Pages for live preview.
